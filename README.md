>项目名称：TMZ，使用非常火的vue框架、webpack工具开发。
>项目描述：第一页面是新闻展示，可下拉刷新，上拉加载，点击新闻列表可查看新闻等，顶部是新闻类型，可选择不同的新闻类型显示不同的新闻列表。
>当然功能还有待完善。



### 项目技术架构
- vue
- vue-router
- vuex
- vue-resource
- vue-cli
- webpack
- muse-ui


### 安装部署
1.下载
```
git clone https://github.com/zlxone/vue-TMZ.git
```
2.进入项目
```
cd vue-TMZ
```
3.安装依赖
```
npm install
```
4.运行
```
npm run dev
```
5.打包
```
npm run build
```
